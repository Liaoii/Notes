# Linux命令

## ls - 显示指定工作目录下的内容及属性信息

**ls \[选项\] \[文件\]**



| 常用参数 |                                             |
| -------- | ------------------------------------------- |
| -a       | 显示所有文件及目录，包括以 . 开头的隐藏文件 |
| -l       | 使用长格式列出文件及目录信息                |
| -r       | 将文件按英文字母次序的反序进行显示（z - a） |
| -A       | 与 -a 相同，但不列出 . 及 ..                |
| -S       | 根据文件大小从大到小进行排序                |
| -R       | 递归列出所有子目录                          |

| 实例    |                                                  |
| ------- | ------------------------------------------------ |
| ls -lhS | 按照文件大小从大到小显示当前目录下的文件和文件夹 |

## cp – 复制文件或目录

**cp [选项] [源文件名] [目标文件名]**

| 常用参数 |                                                              |
| -------- | ------------------------------------------------------------ |
| -f       | 若目标文件已存在，直接覆盖原文件                             |
| -i       | 若目标文件已存在，会询问是否覆盖（与不加参数时的效果一样）   |
| -p       | 保留源文件或目录的所有属性                                   |
| -r       | 递归复制文件或目录                                           |
| -d       | 当复制符号连接时，把目标文件或目录也建立为符号连接，并指向与源文件或目录连接的原始文件或目录 |
| -l       | 对源文件建立硬连接，而非复制文件                             |
| -s       | 对源文件建立符号连接，而非复制文件                           |
| -b       | 覆盖已存在的文件目标前将目标文件备份                         |
| -v       | 详细显示cp命令执行的操作过程                                 |
| -a       | 等价于 -dpr 选项                                             |
